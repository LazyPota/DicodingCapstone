# Fast API Python

## Installation

1. Make virtual environment:
   ```
   python -m venv env
   ```
   
2. Using virtual environment:
   ```
   venv\Scripts\Activate.ps1
   ```

3. Install depedencies:
   ```
   pip install -r requirements.txt
   ```

4. Running the application:
   ```
   uvicorn main:app --reload
   ```
